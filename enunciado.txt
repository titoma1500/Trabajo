Hacer un inicio de sesión que lleve a un formulario y guarde las respuestas para que luego el administrador tenga un tipo de reporte sobre las respuesta enviadas

Para esto hacer un formulario de creación de cuenta, de inicio de sesión y el formulario, y para el administrador un logeo que le muestre todos los comentarios
y una opción de filtrar por id del usuario que lo puso

nombre de la db prueba:
nombre de la tabla de usuarios: usuario
campos: id_usuario (primary key), nombre_usuario, rol

tabla de comentarios: 
nombre: comentario
columnas: id_comentario (primary key), comentario, id_usu_comen 

no es necesario hacer controles, el rol por default al crear el usuario es 'user'
Si el rol del usuario es 'admin' (el cual ya se insertará a la base desde xampp mismo, no desde el sistema) debe ver los comentarios, por quien se hizo y debe poder eliminarlos además de poder buscarlos por id del usuario

el archivo base debe ser index.php, luego hay una carpeta llamada models, donde deberá ir la conexión y los modelos necesarios para las operaciones y hay una view de usuario y luego hay otra carpeta llamada views, donde irá la de admin y la de user con sus respectivas funciones
